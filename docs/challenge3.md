### Challenge 3: Create Multi-agent system

**Introduction:**

Bring your prompts and tools together in the visual designer to create a fully functional multi-agent system.

**Description:**

Initialize a "Multi-step agent" in Agent Designer. Use the initial chat pane to generate the parent-child structure, then paste your crafted prompts into the respective blocks (Main Agent and Data Insight Agent).

**Success Criteria:**

* A Multi-step agent is successfully created in the UI.
* Both the main and sub-agents contain their respective configurations and tools.
* The routing logic seamlessly connects the main agent to the sub-agent for analytical queries.

**Learning Resources:**

* [Google Cloud Docs: Create a multi-step agent](https://docs.cloud.google.com/gemini/enterprise/docs/agent-designer/create-agent)

**Tips:**

* Leverage the initial generation prompt to have Gemini build the boilerplate structure for you automatically.
